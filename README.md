# music_search
* music_search frontend using ReactJS App: on http://localhost:3000/

## Set up
- Run the following commands to download the repo and install dependencies:

    ```
    $ git clone https://github.com/Stephanie1226/music_search.git
    $ cd music_search
    $ npm install
    $ npm start
    ```
